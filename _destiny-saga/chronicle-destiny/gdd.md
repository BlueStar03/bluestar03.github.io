### 3.2 Controls
Explain the control scheme for the game.

#### Gamepad Controls
- **Left Stick**: Move character
- **Right Stick**: Control camera
- **A Button**: Jump
- **B Button**: Attack
- **X Button**: Interact
- **Y Button**: Use item
- **Left Trigger**: Aim
- **Right Trigger**: Shoot
- **Start Button**: Pause/Menu

#### Keyboard Controls
- **WASD Keys**: Move character
- **Mouse**: Control camera
- **Spacebar**: Jump
- **Left Mouse Button**: Attack
- **E Key**: Interact
- **Q Key**: Use item
- **Right Mouse Button**: Aim
- **R Key**: Reload
- **Esc Key**: Pause/Menu

#### Touch Controls
- **Virtual Joystick**: Move character
- **Swipe Screen**: Control camera
- **Tap Button**: Jump
- **Tap Button**: Attack
- **Tap Button**: Interact
- **Tap Button**: Use item
- **Tap and Hold**: Aim
- **Tap Button**: Shoot
- **Tap Menu Icon**: Pause/Menu
